Política de Privacidade - Guia de Entretenimento

Coleta de Dados

O aplicativo coleta informações sobre os canais de TV que você assiste para melhorar a experiência do usuário.

Uso de Dados

Os dados coletados são usados para personalizar as recomendações de canais e melhorar o aplicativo.

Cookies

O aplicativo utiliza cookies para rastrear informações anônimas sobre o uso, a fim de aprimorar os serviços oferecidos.

Anúncios da Unity

Exibimos anúncios da Unity para financiar a operação do aplicativo. A Unity pode coletar dados para exibir anúncios relevantes.

Atenção
